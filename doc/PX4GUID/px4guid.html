使用QGroundControl查看飞控序列号

打开 QGroundControl， 找到 Analyze Tools 选项，进入后打开 MAVLink Console。

输入以下命令查看飞控信息

ver all

MAVLink Console 输出如下

HW arch: HOLYBRO_KAKUTEH7
FW git-hash: 4649f6672ca6d7150ee2d5a43ff2f9db69353047
FW version: 1.13.3 0 (17629952)
FW git-branch: master_v1.13.3
OS: NuttX
OS version: Release 11.0.0 (184549631)
OS git-hash: 4a1dd8680cd29f51fb0fe66dcfbf6f69bec747cf
Build datetime: Sep 20 2025 19:00:46
Build uri: localhost
Build variant: default
Toolchain: GNU GCC, 9.3.1 20200408 (release)
PX4GUID: 00060000000038303334343051060034001c
MCU: STM32H7[4|5]xxx, rev. V

这里的PX4GUID: 00060000000038303334343051060034001c就是飞控序列号。

有些飞控的序列号是不能通过 ver all 命令查看的，比如pixhauk飞控。这些飞控通常会直接广播出来，在QGC的日志中可以看到。
